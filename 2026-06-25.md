## 1. Moebius: Serving Mixture-of-Expert Models with Seamless Runtime Parallelism Switch

> Moebius：支持运行时并行策略无缝切换的 MoE 模型服务系统

- **arXiv ID:** 2606.26607v1
- **Authors / 作者:** Shaoyu Wang, Yizhuo Liang, Jaeyong Song, Chong Li, Seo Jin Park
- **Published / 发布日期:** 2026-06-25
- **Categories / 分类:** cs.DC
- **arXiv URL:** http://arxiv.org/abs/2606.26607v1

### Abstract / 摘要

Mixture-of-Experts (MoE) architectures scale large language models (LLMs) to hundreds of billions of parameters. Serving a single MoE model requires multiple GPUs operating in parallel, typically through tensor parallelism (TP) or expert parallelism (EP). The optimal choice depends on the number of in-flight requests: TP is faster at low concurrency, whereas EP wins at high concurrency. Production workloads cross this boundary continually: online serving sees bursty arrivals that subside into quiet periods, and reinforcement-learning rollouts begin as a high-concurrency burst that decays into a long tail of stragglers. Pinning either layout therefore forfeits performance when the workload crosses to the other side. We present Moebius, a serving system that switches between EP and TP at runtime without restarting the engine or dropping in-flight requests. Our key insight is that EP and TP are two layouts of one model, not two models: they compute the same function over byte-identical expert weights and KV cache, so a switch changes only which rank owns each slice. Moving those owner-changed slices is the sole irreducible cost, and modern high-bandwidth GPU interconnects make it fast enough to do between decode steps without draining in-flight requests. Moebius preserves each parallelism's runtime resident, and reshards the single copy of expert weights and KV cache at fixed addresses with fused GPU-to-GPU transfer kernels. On 8x H200 GPUs serving Qwen3-235B-A22B, Moebius matches the better static parallelism at every operating point, and beats it on RL rollouts by 1.16-1.25x across steps. Each switch completes in 215-434 ms, and Moebius holds both layouts resident with only 2.4% memory overhead.

混合专家（MoE）架构将大语言模型扩展到数千亿参数。服务单个 MoE 模型需要多块 GPU 并行工作，通常通过张量并行（TP）或专家并行（EP）。最优选择取决于在途请求数量：低并发时 TP 更快，高并发时 EP 更优。生产工作负载不断跨越这一边界：在线服务经历突发到达后进入安静期，强化学习 rollout 以高并发突发开始，然后衰减为长尾 stragglers。固定任一布局都会在工作负载跨越到另一边时损失性能。本文提出 Moebius，一种 serving 系统，可在运行时无需重启引擎或丢弃在途请求即可在 EP 和 TP 之间切换。我们的核心洞察是 EP 和 TP 是同一模型的两种布局，而非两个模型：它们对完全相同的专家权重和 KV cache 计算相同函数，因此切换只改变每个切片由哪个 rank 持有。移动这些所有权变更的切片是唯一的不可约成本，现代高带宽 GPU 互联使其快到可以在解码步骤之间完成，无需排空在途请求。Moebius 保留每种并行策略的运行时驻留状态，并通过融合的 GPU-to-GPU 传输内核在固定地址重新分片专家权重和 KV cache 的单一副本。在 8x H200 GPU 上服务 Qwen3-235B-A22B 时，Moebius 在每个工作点都匹配更好的静态并行策略，在 RL rollout 上每步提升 1.16-1.25 倍。每次切换在 215-434 ms 内完成，Moebius 以仅 2.4% 的内存开销同时驻留两种布局。

## 2. EntMTP: Accelerating LLM Inference with Entropy Guided Multi Token Prediction

> EntMTP：基于熵引导的多 token 预测加速大语言模型推理

- **arXiv ID:** 2606.27550v1
- **Authors / 作者:** Carrie Chen
- **Published / 发布日期:** 2026-06-25
- **Categories / 分类:** cs.CL, cs.LG
- **arXiv URL:** http://arxiv.org/abs/2606.27550v1

### Abstract / 摘要

Multi-token prediction has been shown to increase data density during training, improve downstream text-generation quality, and serves as the defacto approach for self-speculative decoding. Existing foundation and open source models that use MTP heads commit to a static tree-based attention topology throughout the entire generation sequence, meaning the speculation depth, and thus the compute required during verification, stays constant regardless of the context. This is fundamentally misaligned with the entropy patterns of natural language where low-entropy regions often support reliable multi-step drafting, while high-entropy regions require more conservative speculation. To address this, we propose Entropy-guided Multi-Token Prediction (EntMTP), a training-free scheduler that toggles between tree-based attention topologies from a set of task-specific pareto-optimal trees conditioned on a running estimate of local generation entropy. By matching speculation depth to context predictability, EntMTP maximizes expected accepted-token throughput across the full distribution of generated text without sacrificing generation quality. When evaluated across Humaneval, ShareGPT, GSM8k, and Litbench benchmarks, EntMTP consistently achieves a 1.15x speedup against Hydra and peak speedup of 1.36x against Medusa baselines respectively.

多 token 预测已被证明能够提升训练期间的数据密度、改善下游文本生成质量，并已成为自投机解码的事实标准方法。现有的基础模型和开源模型使用的 MTP head 在整个生成序列中采用静态的树状注意力拓扑，即无论上下文如何，投机深度以及验证阶段所需计算量都保持不变。这与自然语言的熵模式存在根本性错位：低熵区域通常支持可靠的多步起草，而高熵区域则需要更保守的投机。为此，我们提出熵引导多 token 预测（EntMTP），一种无需额外训练的调度器，它根据局部生成熵的运行估计，在一组任务特定的帕累托最优树之间切换树状注意力拓扑。通过将投机深度与上下文可预测性相匹配，EntMTP 在不牺牲生成质量的前提下，最大化生成文本全分布上的期望接受 token 吞吐量。在 HumanEval、ShareGPT、GSM8k 和 Litbench 基准上的评估表明，EntMTP 相对 Hydra 一致取得 1.15 倍加速，相对 Medusa 基线峰值加速达 1.36 倍。

## 3. SharQ: Bridging Activation Sparsity and FP4 Quantization for LLM Inference

> SharQ：面向 LLM 推理的激活稀疏性与 FP4 量化联合优化

- **arXiv ID:** 2606.26587v1
- **Authors / 作者:** Haoqian Meng, Yilun Luo, Yafei Zhao, Wenyuan Liu, Huaqing Zheng, Xindian Ma, Peng Zhang
- **Published / 发布日期:** 2026-06-25
- **Categories / 分类:** cs.LG, cs.AI
- **arXiv URL:** http://arxiv.org/abs/2606.26587v1

### Abstract / 摘要

Low-bit floating-point formats and semi-structured sparsity are increasingly supported by modern accelerators, yet combining them for LLM activation compression remains challenging: activations contain input-dependent outliers that dominate block scales in FP4 quantization, and directly applying N:M sparsity masks discards moderate values, coupling sparsification loss with quantization error. We introduce SharQ, a training-free inference method that bridges activation sparsity and FP4 quantization through an online sparse--dense decomposition. For each activation tensor, SharQ generates an input-adaptive N:M mask to extract an outlier-dominated sparse backbone, quantizes it to FP4, and defines a dense residual relative to the quantized sparse backbone rather than the unquantized sparse values. A sparse FP4 GEMM processes the backbone while a dense FP4 GEMM compensates for both mask-induced activation loss and sparse-path quantization error. The two paths share a single FP4 weight payload with path-specific scale views, and a fused preparation kernel absorbs mask generation, residual construction, and layer normalization into one operator. SharQ requires no calibration data, retraining, or model-specific tuning. Evaluated on Llama-3.1-8B, Qwen2.5-7B, Qwen3-30B-A3B, and Qwen3-VL-8B, SharQ recovers 43--63% of the NVFP4-to-FP16 accuracy gap across language and vision-language tasks, and generalizes across NVFP4, HiF4, and MXFP4 formats. On an RTX 5090, SharQ delivers 2.2--2.4× latency reduction over FP16 and 1.2--1.4× throughput improvement over FP8 in language model serving, and up to 1.58× speedup on Wan2.2-T2V-A14B video generation when combined with SageAttention.

现代加速器越来越支持低比特浮点格式和半结构化稀疏性，但将两者结合用于 LLM 激活压缩仍具挑战：激活包含与输入相关的异常值，这些异常值在 FP4 量化中主导块尺度；直接应用 N:M 稀疏掩码会丢弃中等值，使稀疏化损失与量化误差耦合。本文提出 SharQ，一种无需训练推理方法，通过在线稀疏-稠密分解桥接激活稀疏性与 FP4 量化。对于每个激活张量，SharQ 生成输入自适应的 N:M 掩码以提取异常值主导的稀疏骨干，将其量化为 FP4，并定义相对于量化后稀疏骨干的稠密残差。稀疏 FP4 GEMM 处理骨干，稠密 FP4 GEMM 补偿掩码引起的激活损失和稀疏路径量化误差。两条路径共享单一 FP4 权重负载，具有路径特定的尺度视图，融合准备核将掩码生成、残差构建和层归一化吸收为一个算子。SharQ 无需校准数据、重训练或模型特定调优。在 Llama-3.1-8B、Qwen2.5-7B、Qwen3-30B-A3B 和 Qwen3-VL-8B 上的评估表明，SharQ 在语言和视觉-语言任务上恢复了 NVFP4 到 FP16 精度差距的 43%-63%，并可泛化到 NVFP4、HiF4 和 MXFP4 格式。在 RTX 5090 上，SharQ 在语言模型 serving 中相比 FP16 降低 2.2-2.4 倍延迟、相比 FP8 提升 1.2-1.4 倍吞吐，结合 SageAttention 时在 Wan2.2-T2V-A14B 视频生成上最高加速 1.58 倍。

## 4. CAT-Q: Cost-efficient and Accurate Ternary Quantization for LLMs

> CAT-Q：面向大语言模型的高性价比精确三值量化

- **arXiv ID:** 2606.26650v1
- **Authors / 作者:** Shigeng Wang, Chao Li, Yangyuxuan Kang, Jiawei Fan, Anbang Yao
- **Published / 发布日期:** 2026-06-25
- **Categories / 分类:** cs.CL, cs.AI
- **arXiv URL:** http://arxiv.org/abs/2606.26650v1

### Abstract / 摘要

In this paper, we present CAT-Q, Cost-efficient and Accurate Ternary Quantization, for compressing and accelerating LLMs. Unlike existing state-of-the-art ternary quantization methods that rely on data-intensive and costly quantization-aware training to mitigate severe performance degradation, CAT-Q is a simple yet effective post-training quantization scheme that is readily applicable to LLMs with diverse architectures and model sizes. It has two key components, learnable modulation (LM) and softened ternarization (ST), which are coupled from an optimization perspective. LM leverages a composition of learnable factors to modulate the distribution of pre-trained high-precision weights and the ternary threshold, making them less sensitive to ternarization. ST further introduces a differentiable transition function to guide the ternarization process toward stable convergence. We show that, for pre-trained LLMs with 1.7B to 8B parameters, CAT-Q can efficiently quantize them into ternary models using only 512 calibration samples, while achieving superior performance than the seminal BitNet 1.58-bit v1 and v2 families (with 1.3B to 7B parameters) trained with 100B tokens, yielding about a 100,000X reduction in training tokens. Moreover, we show for the first time that CAT-Q can quantize much larger pre-trained LLMs having 14B to 235B parameters into leading ternary models within just 8 to 60 hours on 8 A100-80GB GPUs.

本文提出 CAT-Q（Cost-efficient and Accurate Ternary Quantization），用于压缩和加速大语言模型。与现有最先进的三值量化方法不同，那些方法依赖数据密集且昂贵的量化感知训练来缓解严重性能退化，而 CAT-Q 是一种简单有效的训练后量化方案，可直接应用于不同架构和模型规模的 LLM。它有两个关键组件：可学习调制（LM）和软化三值化（ST），两者从优化角度耦合。LM 利用一组可学习因子来调制预训练高精度权重分布和三值阈值，使它们对三值化不那么敏感。ST 进一步引入可微过渡函数来引导三值化过程稳定收敛。我们表明，对于 1.7B 到 8B 参数的预训练 LLM，CAT-Q 仅使用 512 个校准样本即可高效将其量化为三值模型，同时性能优于使用 100B token 训练的 BitNet 1.58-bit v1 和 v2 系列（1.3B 到 7B 参数），训练 token 减少约 10 万倍。此外，我们首次表明 CAT-Q 可以在 8 张 A100-80GB GPU 上仅用 8 到 60 小时将 14B 到 235B 参数的更大预训练 LLM 量化为领先的三值模型。

## 5. HyperDFlash: MHC-Aligned Block Speculative Decoding with Gated Residual Reduction

> HyperDFlash：面向 MHC 架构的门控残差压缩块投机解码

- **arXiv ID:** 2606.26744v1
- **Authors / 作者:** Luxi Lin, Shuang Peng, Rui Ma, Junhao Hua, Shuwei Fan, Zhengda Qin, Qiang Wang, Hongjian Sun, Fangmin Chen, Songwei Liu
- **Published / 发布日期:** 2026-06-25
- **Categories / 分类:** cs.LG, cs.CL
- **arXiv URL:** http://arxiv.org/abs/2606.26744v1

### Abstract / 摘要

We present HyperDFlash, a block-parallel speculative decoding framework tailored to the novel multi-hyper-connection (MHC) architecture proposed by DeepSeek-V4. Despite the strong initial-token drafting performance of the native Multi-Token Prediction (MTP) module in DeepSeek-V4, its draft accuracy degrades sharply at later positions, as error accumulation from unverified intermediate tokens harms acceptance rates. Although the original DFlash method supports efficient one-pass block drafting, it cannot be seamlessly adapted to the MHC paradigm, since the multi-path residual stream of DeepSeek-V4 induces feature misalignment with conventional drafting designs. To resolve this mismatch, we propose two model-aligned optimizations for MHC residual streams. First, we adopt pre-collapse residual states as the exclusive conditioning signal, preserving multi-path structural information and aligning the drafter with the native prediction pathway of the target model. Second, we replace the heavy generic linear compressor with a lightweight gated residual reducer, whose parameters are inherited from the built-in hyper-connection head. This design yields input-aware path aggregation with three orders of magnitude fewer parameters while maintaining architectural alignment. We further enhance training via a targeted KL distillation loss applied to the LM-head, which regularizes predictions against the full target probability distribution and improves draft quality at early training stages. Experiments across math reasoning, code synthesis, and conversational benchmarks show that HyperDFlash consistently outperforms both the native MTP baseline and vanilla DFlash adaptation. It achieves substantial gains in average accepted draft length and decoding speedup, validating the effectiveness of MHC alignment, gated reduction, and targeted distillation for high-performance speculative decoding.

本文提出 HyperDFlash，一种针对 DeepSeek-V4 提出的新型多超连接（MHC）架构定制的块并行投机解码框架。尽管 DeepSeek-V4 原生多 token 预测（MTP）模块在初始 token 起草上表现出色，但其草稿准确性在后续位置急剧下降，因为来自未验证中间 token 的误差积累损害了接受率。虽然原始 DFlash 方法支持高效的一次性块起草，但它无法无缝适配 MHC 范式，因为 DeepSeek-V4 的多路径残差流导致与传统起草设计的特征不对齐。为解决这一不匹配，我们提出两种针对 MHC 残差流的模型对齐优化。首先，我们采用预折叠残差状态作为唯一的条件信号，保留多路径结构信息，并使起草器与目标模型的原生预测路径对齐。其次，我们用轻量级门控残差压缩器替代沉重的通用线性压缩器，其参数继承自内置超连接头。该设计以少三个数量级的参数实现输入感知路径聚合，同时保持架构对齐。我们通过针对 LM-head 的目标 KL 蒸馏损失进一步增强训练，使预测相对完整目标概率分布正则化，并改善早期训练阶段的草稿质量。在数学推理、代码合成和对话基准上的实验表明，HyperDFlash 始终优于原生 MTP 基线和普通 DFlash 适配。它在平均接受草稿长度和解码加速上取得显著提升，验证了 MHC 对齐、门控压缩和目标蒸馏对高性能投机解码的有效性。

## 6. Epiphany-Aware KV Cache Eviction Without the Attention Matrix

> 无需注意力矩阵的顿悟感知 KV Cache 驱逐

- **arXiv ID:** 2606.26472v1
- **Authors / 作者:** Steven Kolawole, Virginia Smith
- **Published / 发布日期:** 2026-06-25
- **Categories / 分类:** cs.LG, cs.CL
- **arXiv URL:** http://arxiv.org/abs/2606.26472v1

### Abstract / 摘要

As reasoning models emit chains of thought tens of thousands of tokens long, KV cache increasingly becomes a deployment bottleneck. Existing cache eviction methods rank tokens by attention weight, which is a noisy importance proxy in long reasoning traces, and prohibits the use of fused kernels in production inference by forcing the model to materialize the attention matrix. In this work, we instead score tokens with a metric we term the epiphany score: the change in the model's internal representation, read directly from the forward pass with no attention matrix and negligible extra state. Our resulting cache eviction method, EpiKV, requires no training, classifier, or custom kernel, and can be used directly in FlashAttention inference stacks unchanged -- scaling to a 16x longer feasible context than attention-based scoring. At a 4096-token cache EpiKV reaches 72% on MATH-500, matching the strongest attention-based baseline (ThinKV 71%, H2O 67%); a lag-normalized KV variant reaches 37% on AIME-2024 at 8192 tokens against the best of them (33%), at up to 2.8x the speed.

随着推理模型生成数万 token 长的思维链，KV cache 日益成为部署瓶颈。现有的 cache 驱逐方法按注意力权重对 token 排序，这在长推理轨迹中是一种噪声较大的重要性代理，并强制模型物化注意力矩阵，从而禁止了生产推理中融合内核的使用。在本文中，我们改用一种称为顿悟分数（epiphany score）的指标为 token 打分：它衡量模型内部表示的变化，可直接从前向传播读取，无需注意力矩阵，额外状态可忽略不计。我们提出的 cache 驱逐方法 EpiKV 无需训练、分类器或自定义内核，可直接在未经修改的 FlashAttention 推理栈中使用——可将可行上下文扩展到基于注意力打分的 16 倍。在 4096 token cache 下，EpiKV 在 MATH-500 上达到 72%，与最强基于注意力的基线持平（ThinKV 71%，H2O 67%）；一种滞后归一化 KV 变体在 AIME-2024 的 8192 token 上达到 37%，优于其他方法（33%），速度最高快 2.8 倍。

## 7. Position Rebinding Cache Reuse: Replay-Free Visual Revisiting for Interleaved Multimodal Reasoning

> 位置重绑定 Cache 重用：面向交错多模态推理的无重放视觉回访

- **arXiv ID:** 2606.26631v1
- **Authors / 作者:** Mengzhao Wang, Yanli Ji, Wangmeng Zuo, Peng Ye, Chongjun Tu
- **Published / 发布日期:** 2026-06-25
- **Categories / 分类:** cs.CV
- **arXiv URL:** http://arxiv.org/abs/2606.26631v1

### Abstract / 摘要

Interleaved multimodal reasoning improves visual grounding by revisiting visual evidence during multi-step generation, yet existing methods typically rely on token replay, repeatedly forwarding selected visual tokens. A natural shortcut is to reuse the historical visual key-value (KV) cache directly. However, we identify a critical failure mode of this strategy: cached visual keys are already bound to their original positional context. Such stale positional binding distorts attention under later decoding contexts and can trigger severe autoregressive decoding collapse. This failure suggests that effective cache reuse requires reconstructing visual evidence under positions compatible with the current decoding state, rather than directly copying position-bound historical cache entries. To this end, we propose Position Rebinding Cache Reuse (PRCR), a cache-level framework for replay-free visual revisiting. PRCR stores raw visual KV cache together with their original spatial coordinates, then reassigns position-compatible coordinates to select entries and rebinds their keys before injecting the reconstructed cache into the active decoder cache. This design reuses historical visual evidence while preserving textual positional continuity and relative visual structure. Experiments across multiple multimodal reasoning benchmarks show that PRCR achieves replay-level or better performance, improving average accuracy by 5 percent and reducing visual-revisiting computation by up to tens of thousands of times.

交错多模态推理通过在多步生成中回访视觉证据来改善视觉定位，但现有方法通常依赖 token 重放，反复前向传播选定的视觉 token。一个自然的捷径是直接重用历史视觉键值（KV）cache。然而，我们发现该策略的一个关键失败模式：缓存的视觉 key 已经绑定到其原始位置上下文。这种陈旧的位置绑定会在后续解码上下文中扭曲注意力，并可能触发严重的自回归解码崩溃。这一失败表明，有效的 cache 重用需要在兼容当前解码状态的位置下重建视觉证据，而不是直接复制位置绑定的历史 cache 条目。为此，我们提出位置重绑定 Cache 重用（PRCR），一种面向无重放视觉回访的 cache 级框架。PRCR 存储原始视觉 KV cache 及其原始空间坐标，然后为选定条目重新分配兼容位置的坐标，并在将重建的 cache 注入活动解码器 cache 之前重新绑定其 key。该设计在保留文本位置连续性和相对视觉结构的同时重用历史视觉证据。在多个多模态推理基准上的实验表明，PRCR 达到重放级或更好的性能，平均准确率提升 5%，并将视觉回访计算减少多达数万倍。

## 8. Information-Aware KV Cache Compression for Long Reasoning

> 面向长推理的信息感知 KV Cache 压缩

- **arXiv ID:** 2606.26875v1
- **Authors / 作者:** Jushi Kai, Zhuiri Xiao, Alexandra Birch, Zhouhan Lin
- **Published / 发布日期:** 2026-06-25
- **Categories / 分类:** cs.CL, cs.AI
- **arXiv URL:** http://arxiv.org/abs/2606.26875v1

### Abstract / 摘要

Reasoning capability has advanced rapidly in large language models (LLMs), leading to an increasing size of key-value (KV) cache in both prefilling and decoding stages. Existing KV cache compression methods mainly rely on attention weights to estimate token importance. While attention effectively captures contextual relevance, it overlooks complementary information-theoretic signals related to predictive uncertainty and token informativeness. In this paper, we revisit token importance from a forward-looking perspective and introduce Forward Influence, a metric that measures how compressed tokens affect future contexts. Our analysis reveals that tokens selected by attention scores mainly influence nearby contexts, whereas tokens associated with high predictive uncertainty exhibit substantially stronger influence on distant future contexts. Based on the observation, we propose InfoKV, an entropy-aware KV cache compression framework that incorporates information-theoretic signals. It combines token-level predictive uncertainty with layer-wise representation evolution and integrates the resulting entropy scores with attention scores during reasoning. Experiments on long-context reasoning benchmarks with Llama-3.1, Llama-3.2, and DeepSeek-R1 demonstrate that InfoKV consistently outperforms existing attention-based KV compression methods in both long prefilling and decoding scenarios.
